Overview
--------

A Python library for Dataflow Analyzer that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install dfa_lib_python


Or clone the repo:

    $ git clone https://github.com/viniciusscampos/dfa_lib_python.git
    $ python setup.py install
