Documentation of RESTful services

Provenance Data Extractor


Methods for prospective provenance data

URL: /pde/dataflow

Concept Method (body of the HTTP request)
Dataflow dataflow(tag)
Program program(name, path)
Dataset dataset(tag, attributes, types)
Extractor extractor(datasetTag, tag, method, cartridge, namesOfAttributes, typesOfAttributes)
Extractor Combination extractor_combination(datasetTag, innerExtractorTag, outerExtractorTag, namesOfAttributes, typesOfAttributes)
Transformation transformation(tag, tagsOfInputDatasets, tagsOfOutputDatasets, namesOfPrograms)


Methods for retrospective provenance data

URL: /pde/task

Concept Method (body of the HTTP request)
Task task(dataflowTag, transformationTag, ID[, subID], status[, workspace, resource])
File file(name, path)
Collection collection(datasetTag, elements)
Dependency dependency(transformationTags, taskIDsOfDependentTasks)
Query Interface


Methods for running a query considering a fragment of the dataflow path

URL: /query_interface/{dataflow_tag}/{dataflow_version}

Concept Method (body of the HTTP request) Additional information
Mapping mapping(type) type = PHYSICAL, LOGICAL or HYBRID
Source datasets source(datasetTags)
Target datasets target(datasetTags)
Includes include(datasetTags) Datasets to be included in the fragment of dataflow path.
Excludes exclude(datasetTags) Datasets to be excluded from the fragment of dataflow path.
Projections projection(attributes) attributes argument defines which attributes will be obtained after query processing.
e.g., attributes = {table1.att1;table2.att2}
Selections selection(conditions) conditions is used to filter only relevant data elements.
e.g., table1.att1 > 100
